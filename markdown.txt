# Markdown sample 🌟

Markdown is a simple and flexible way to create rich text documents. Here's a quick tour of some features!

---

## Code Examples 👨‍💻

Inline code is handy for small snippets like `console.log('Hello, World!');`.

For larger code blocks, use triple backticks:

```javascript
// An example of a function with a default parameter
function greet(name = "World") {
    console.log(`Hello, ${name}!`);
}

greet();       // Outputs: Hello, World!
greet("Alice"); // Outputs: Hello, Alice!
```

---

## Text Styling ✍️

Emphasize words with **bold**, *italics*, or even ***both***.

Strikethroughs are also supported: ~~mistake~~ or ~typo~.

---

Add quotes or poetic excerpts with a `>`:
